## vscode 编辑器

1. 打开控制面板（⇧⌘P)
2. 输入 shell command 
3. 在提示里看到 Shell Command: Install ‘code’ command in PATH， 就可以了

![](editor/twst.png)

### 在命令行你需要打开的目录下运行
```code
code .
```
