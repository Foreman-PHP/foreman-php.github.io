镜像命令

docker images 查看主机上所有的镜像

  1. 解释
  • REPOSITORY 镜像的仓库源
  • TAG 镜像的标签
  • IMAGE ID 镜像的ID
  • CREATED 镜像的创建时间
  • SIZE 镜像的大小
  1. 可选项

docker search 搜索镜像

  1. 可选项 -f, 过滤

docker pull 拉去镜像

docker rmi 删除镜像

docker run 新建容器并启动
